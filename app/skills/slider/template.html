Java

Java is an Object Oriented type language developed by Sun Microsystems. It was created in 1995, designed to run independently of the plataform it was developed in and runned at. This is achieved by pre-compiling the source code into bytecode, providing another level of security against intelectual theft and allowing abstraction from computer architecture, since the JVM( Java Virtual Machine ), will thereafter interpret the bytecode and convert it into machine code specific to that machine.

Due to this, Java has gained a lot of traction and it is currelty used in a variety of projects, from simple aplications to operating systems. It's mainstream use generated lots of documentation along with tools that augment it's grasp, allowing Java to be compatabile with a huge number of tecnologies.

Html 5

HyperText Markup Language is the standart markup language for web pages and web aplications. It's place in the modern world of the web is irrefutable, forming a corner stone for the industry. It can be said that any companie today will have to interact with this tecnology somehow, if it is to explore the online market.

Having extensive documentation and support online, many tecnologies have adapted to somehow interact with it and even disregard some functionalities, replacing them with Html templates.

Html is a solid tecnology to know about and it will stay in the top programing languages for a long time to come.

Css 3

Combined with Html, Cascading Style Sheet provides the web with asteticaly pleasing content. It allows the styling of all Html elements, coding their presentation and layout, along with responsive web design for all devices.

Today Css can be obtained with many pre-written libraries, setting Twitter's Bootstrap as one of the major examples, along with online error documentation and support.

Projected by the World Wide Web dependency, the spread of mobile devices and cost reduction on personal computeres and laptops have made this tecnology key to user experience, making it essensial for any company looking to enter the online market provided by modern comunication devices.

Javascript

Javascript is a phenomenon of current times. Created as a small language to be ran on the browser as a sidekick to Java, its slick sintax and genius implementation quickly propeled it to one of the most used programing language of current times.

Javascript has become the first lambda language to go mainstream, as milions of developers have learned to embrace the concept of dynamic types, closure, prototypical inheritance and first-class functions, allowing for a new level of expression, in a beautifull, simple, non-verbose way.

Today, there are dozens of libraries and frameworks developed arround Javascript that, by themselves, have already left their mark in software development.

AngularJs

Angular Js is one of the many frameworks that derived from the Javascript explosion in the last couple of years. By combining a set of libraries into a single aplication, it provides a multitude of functionality on top of Html content, making the development of single web applications easier to code, organize and deploy.

A newer version of this framework is currently live. Angular 2 has grown outside the spectrum of the browser, following the trend of Javascript, and now reaches back-end and desktop functionality.

Given the time necessary for the industry of software to adjust to new tools, AngularJs will still be arround for a good amount of years to come allowing the migration process to take place.

MySQL

Developed by Oracle, it is a Relational Database Managment System, distributed freely and open source. It is an amazing tool to store in a persistent, compact and consistent form, huge amounts of data.

Due to it's open-source nature and quality, it is a great solution for many systems that require data storage and consumption, as most companies do for inventory and client managment, without implying an increased overhead cost on the company.

It provies an exelent plataform for database design with logical schemas along side good compatibility with other tecnologies, a good engine and account managment for system security.

UML

Unified Modeling Language dates back to before 1990, where it was proposed as a method for software engineers to describe a project through graphical documentation.

Although the problem of translating software architecture in a clear conscise way in order to facilitate the development of said project is a problem that has persisted throughout the history of software engineering, the fact that this modeling language hasn't gained a spot in many companies proves still that the industry of software development has a long way to go before it reaches the maturity of other fields such as architecture or automobile.

C Language

This imperative programing language, combined with C++, is one of the most used low level languages. C allows programers access to memory control with a low level of abstraction from hardware.

Despite its replacement by languages with a higher level of abstraction, C's sintax has been imitated numerous times, as well as some of its features. This makes the language relevant to this day, as it provides insite on how to write clean and efficient code.

C has been used to contruct other languages, engines and operating systems, and it continues to be the go to language when dealing with the lack of processing power or hardware interactivity.
